# Naija Mart — HMG StoreForge v3 DEMO STORE

This is a **fully-populated demo** showing every v3 feature: 12 products with images,
wishlist, reviews, variants, multi-image gallery, coupons (try **WELCOME10**), delivery
zones, store hours, FAQ, testimonials, multi-currency, order tracking, PWA, and more.

## View it
Open `index.html` in a browser, or deploy this folder to Cloudflare Pages / GitHub Pages.

## Try these
- Add items to cart → Checkout → apply coupon **WELCOME10** → pick a delivery zone.
- Tap ❤️ to save favourites; open a product to leave a ⭐ review.
- Switch currency (NGN/USD/GBP) in the toolbar.
- Visit `track.html` after ordering to track it.
- Open `admin.html` (passcode **admin1234**) → Dashboard, Orders, Bulk CSV.
