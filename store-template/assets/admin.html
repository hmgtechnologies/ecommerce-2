💾 Save & Publish

How publishing works

JSON mode (free, no signup): Click "Download products.json", then upload the file to your GitHub repo (replacing the old one). Your store updates in ~1 minute.

Supabase mode (live, instant): If Supabase is configured in config.js, click "Sync to Supabase" and changes appear instantly with no re-upload.

Full backup: downloads products + reviews + orders + wishlist as one JSON you can restore later or on another device.

❓ Quick help

Full guides are in the docs/ folder of your project. Key points:

  • Add products: Products tab → fill form → Save → Publish tab → Download products.json → upload to GitHub.
  • Images: Use Google Drive links shared as "Anyone with the link".
  • Bulk: Bulk CSV tab → download template → fill in Excel/Sheets → import.
  • Orders/Reviews: stored per device; enable Supabase for centralised storage.
  • Discounts & delivery zones: set in assets/js/config.js (see DEPLOYMENT/FEATURES docs).
