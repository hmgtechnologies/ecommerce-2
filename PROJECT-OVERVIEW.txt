WHAT THIS IS
------------
A platform that GENERATES complete, ready-to-deploy online stores for clients.
You enter client details once; it outputs a full e-commerce website (catalog,
cart, WhatsApp ordering, payments, admin panel) you upload to GitHub / Cloudflare.
Similar in spirit to Jumia / Jiji / Konga, but a single-vendor store per client.
100% FREE TOOLS. NO PAID AI APIs.

FIRST THINGS TO DO
------------------
1) Open generator/assets/js/brand.js → confirm your WhatsApp + email + links.
2) Open generator/index.html in a browser → try generating a test store.
3) Read docs/DEPLOYMENT.md → deploy the generator and your first client store.

THREE-STEP WORKFLOW (per client)
--------------------------------
1. GENERATE: open generator → fill form → Generate → download ZIP (unzip).
   (or: cd generator/cli && node generate.js client.json)
2. DEPLOY:   create GitHub repo named <storeId> → upload files →
             Cloudflare Pages → live at https://<storeId>.pages.dev
3. HANDOVER: send client their store link, admin link, passcode + CLIENT-GUIDE.
